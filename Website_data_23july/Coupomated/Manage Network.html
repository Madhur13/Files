Manage Affiliate Network Relationships

Enter your network IDs and hit the submit button at the bottom of the page. Use Extra Param for tracking Cashback or Additional Analytics, we'll provide variable as mention, you can then replace those variables with dynamic value. ( Ex: Ser Extra Param 1 as  {USERID} and replace it dynamically with Actual UserID in your system.)

IMPORTANT NOTES - PLEASE GO THRU THEM CAREFULLY

  • Double check that data entered are correct as affiliate deep links will be generated using your entered IDs.
  • For few affiliate networks, like Optimise, you need to approve each merchant level program to enable deep linking. Make ensure to get approved first among multiple program for each stores. If this is not done, your links will not work properly.
  • Data once saved, can't be changed/modifed. Enter values carefully. Contact support if you want it changed.
  • After adding the ID's , follow below steps
  1. Set Network Preference and Generate Matrix
  2. Verify Merchant - Network Matrix
  3. Re-Generate Data or Links by visiting Browse Coupons.
