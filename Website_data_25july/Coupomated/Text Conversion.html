Word Replacement

Unique content is essential for any digital product and therefore, here, you can create unique content which would differentiate from others.

You can set the word replacement for any existing (old) words in the field. Also, you can change the word replacement based on either all merchants or specific to select merchant.

For example, you want to change Coupon Code word to Promo Code for all merchants, select merchant as All Merchants, enter old text as Coupon Code and new text as Promo Code. – This will accordingly update the datafeed by replacing all Coupon Code words to Promo Code.

Similarly, you may set the preference based on a merchant also.

Replacement Testing

You can see how word replacements actually function. In sample text, based on variable set above, enter text string here and click on convert to see it replacing words.

For example, we have set to change Coupon Code to Promo Code. Let’s enter a sentence as follows and click on convert to see it working.

Exclusive cashback coupon code. Paytm is offering extra 5% cashback on calling tablets by using above coupon code.

The text would result like;

Exclusive cashback promo code. Paytm is offering extra 5% cashback on calling tablets by using above promo code.
