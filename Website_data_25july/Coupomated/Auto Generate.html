Network Preference

Set here unique preference for each network. This will automatically select network for each merchants based on your set preference.

For ex. Paytm has accounts with Optimise, Payoom and Vcommission. You set preference as 1 - Optimise, 2 - Vcommission and 3 - Payoom, than, this will automatically route all Paytm links via Optimise as you preferred it over other network.

By selecting preferences, the pain to select networks at each merchant level is eliminated. Later, you can manually edit preferences at Merchant – Network Matrix.

 Networks configured under manage networks would only be available here for you to set preference

IMPORTANT NOTE - After making changes, make sure to click Generate Matrix Button and Verify the Generated Data.
To re-build links as per newly saved setting, visit  Browse Coupons and Re-Generate Links
