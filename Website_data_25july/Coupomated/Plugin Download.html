Plugin Download

Download CConnect Plugin and setting file as per your theme. This plugin is to link Coupomated coupon data feed to your website and migrate data as per set crons.


Plugin Name Download Link Plugin Description
CCloacking Plugin

CCloaking plugin is used for converting all nasty looking affiliate links to readable links. This plugin is compatible with any theme on Wordpress.
It is optional plugin for those themes that don't support link cloaking by default. (Ex: Couponer)

This plugin is not FREE and you need to pay one time cost. Click here to request the plugin.
CConnect Plugin

CConnect is core of Coupmated model. This plugin is very powerful and allows you to periodically create coupons completely automaed.
Plugin also allows you to customize how coupons are imported via Drag Drop feature. 

This is part of Coupoamted2Live service, this plugin is NOT available for FREE.

 
WordPress Importer Plugin

Offical plugin relased by WordPress to categories, stores and more from a WordPress export file.
We will provide WordPress export file to create categories and stores.
  1. In WordPress 2.7 and above you can install plugins directly from the admin area.
  2. Download the plugin to your computer, then log in to your WP admin area and go to Plugins > Add New.
  3. Browse to the plugin file and select it. Then click Install Now and the plugin will be installed shortly.
  4. Soon after installation, you’ll see an option on your Wordpress as CConnect.
  5. Import all setting by downloading relevant setting file compatible with your theme.
